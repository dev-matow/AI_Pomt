## STEP 1 : อ่านข้อมูลจากไฟล์ CSV



ใช้ Python อ่านไฟล์ CSV ที่แนบมา



เงื่อนไข:



* ตัดข้อมูลที่ไม่มีค่าในคอลัมน์ "บทวิเคราะห์" ออก

* ใช้ค่าในคอลัมน์ "ชื่อทำเล" แถวแรกเป็นชื่อ Site

* วิเคราะห์แยกตามคู่แข่งแต่ละรายในคอลัมน์ "คู่แข่ง"

## STEP 2 : Normalize Brand Name



ใช้กฎต่อไปนี้



```python

def brand_label(raw):

    if '7-11' in raw or '7-Eleven' in raw:

        return '7-11'



    if 'Lotus' in raw:

        return "Lotus's go"



    if 'CJM' in raw or 'CJ' in raw:

        return 'CJM'



    return raw.split('_')[0]

```



---



## STEP 3 : Extract Location



```python

def brand_location(raw):

    parts = raw.split('_')

    return parts[1] if len(parts) > 1 else ''

```

## STEP 7 : สร้าง Executive Summary



สร้าง Insight 2–4 Bullet ต่อคู่แข่ง



กฎการสรุป



### กรณี Pattern อันดับ 1



ถ้าเป็น



"ที่ทำงาน – ซื้อ – บ้าน"



สรุปว่า



> ส่วนใหญ่มีพฤติกรรมมาจากที่ทำงานแวะซื้อแล้วกลับเข้าบ้าน



ถ้าเป็น



"บ้าน – ซื้อ – บ้าน"



สรุปว่า



> ลูกค้าส่วนใหญ่มีพฤติกรรมออกจากบ้านมาซื้อแล้วกลับบ้าน



ถ้าเป็น



"บ้าน – ซื้อ – กิจกรรม"



สรุปว่า



> ลูกค้าส่วนใหญ่ออกจากบ้านมาซื้อแล้วเดินทางต่อไปยังจุดกิจกรรม



ถ้าเป็น



"ทางผ่าน"



สรุปว่า



> ลูกค้าส่วนใหญ่เป็นกลุ่มทางผ่านไม่มีต้นทางปลายทางชัดเจน



เพิ่มเติม



* คำนวณสัดส่วนลูกค้าที่เกี่ยวข้องกับ "บ้าน"

* หากเกิน 50% ให้ระบุว่าเป็นฐานลูกค้าในชุมชน

* ระบุกลุ่มลูกค้าหลักที่มีจำนวนมากที่สุด
